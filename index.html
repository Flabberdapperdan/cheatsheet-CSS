Flex-box


Parent Container

Tag Description Standard Options
display Defines a flex container and enables it for al its children flex
flex-direction Establishes the main-axis row(default), row-reverse, column, column-reverse
flex-wrap Allow items to wrap as need.
Items will try to fit in one line by default		 nowrap, wrap, wrap-reverse
flex-flow This is shorthand for flex-direction and flex-wrap properties, which together define the flex container's main and cross axes. The default value is row nowrap
justify-content This defines the alignment along the main axis. flex-start, flex-end, start, end, left, direction, right, center, speace-between, space-around, space evenly
align-items This defines the default behvior for how flex items are laid out along the cross axis on the current line. stretch (default), flex-start, start, self-start, flex-end / end / self-end, center, baseline
align-content This align a flex container's lines within when there is extra space in the cross-axis similar to justif-content. normal, flex-start / start, flex-end, center, space-between, space-around, space-evenly, stretch

Properties for the children

Tag Description Standard Options
order The order property controls the order in which they appear.
By default they appear in source order.		 Default value is 0
flex-grow Defines the ability of a flex-item to grow if necessary. It dictates what amount of the available space inside the flex container the item should take up.
If al items have a value of 1, space is distributed equally. If one item then has a value of 2, this item will get twice as much space. 		Negative numbers are invalid
flex-shrink Defines the ability for a flex item to shrink of nexessary. Negative numbers are invalid. Default value is 1.
flex-basis This defines the default sie of an element before the remaining space is distributed. It can be a length or a keyword. auto, main-size, content
flex Shorthand for flex-grow, flex-shrink, and flex-basis combined. It is recommended that you use this shorthand property rather than set the individual properties. Default value is 0 1 auto
align-self This allows the default alignment to be overridden for individual flex-items Default value is value of parent-container
